### 什么是数组
数组是一种数据类型。（把很多数据装入一个盒子中，用的时候在取出来）

### 为什么使用数组
存储多个值的时候，使用数组，每一个索引对应一个值，用来存储一系列值，方便快捷。

### 定义数组
#### 字面量定义

```js
var arr = [1, 2, 3];
var arr2 = ["1", "2", "3"];
```

#### 对象定义

```js
var  arr  =  new Array(参数);
参数位置一个数值时为数组长度，多个数值时为数组中的元素。
```

### 数组操作
#### length操作
##### 获取长度

```js
数组的长度 = 数组名.length;

var arr = [1, 2, 3,  "5"];
var len = arr.length;
```

##### 修改长度
可以通过修改数组的长度来改变数组中元素的个数，如果改小了，数组从后面删除元素。  
**（伪数组的长度可以修改，但是不能修改里面的元素）**

```js
var arr2 = [1, 2, 3, 4];
arr2.length = 6;
// arr2 变为 [1, 2, 3, 4, undefined, undefined];
arr2.length = 2;
// arr2 变为 [1, 2];
```

##### 获取元素

```js
数组中的指定元素 = 数组名[索引值];

var arr = [1, 2, 3,  "5"];
var a = arr[1]; // a = 2;
var b = arr[3]; // b = "5";
```
